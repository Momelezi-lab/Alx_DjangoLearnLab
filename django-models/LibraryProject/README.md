# LibraryProject

## Overview
This Django project was created as part of the **Introduction to Django** task in the ALX Django Learn Lab.  
It introduces the setup of a Django environment, project creation, and the use of the development server.

## Steps Completed
1. Verified Python installation.
2. Installed Django using pip.
3. Created a new Django project named **LibraryProject**.
4. Ran the Django development server.
5. Viewed the default Django welcome page.

## Key Files
- **manage.py** — A command-line utility for Django project management.
- **settings.py** — Configuration settings for the project.
- **urls.py** — Defines the URL routing of the Django application.
